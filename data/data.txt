Description:
Overview
Medeloan Loan Origination System (LOS) is a comprehensive web-based platform designed to streamline the medical loan application process. The system enables partner organizations (Claim Buddy, MedsCred, and other authorized healthcare entities) to submit medical loan applications on behalf of patients through a structured, user-friendly interface with automated document processing capabilities.
Business Objective
Reduce manual data entry errors, accelerate loan application processing time, and ensure data accuracy and consistency through intelligent document processing and validation mechanisms while maintaining full regulatory compliance and comprehensive audit trails.
Key Features & Capabilities
 Multi-Partner Support
 Support multiple partner organizations with different field requirements and configurations
 Partner-specific workflows and optional/mandatory field handling
 Partner user access management and role-based permissions
 Wizard-Based Application Form
 4-step sequential wizard interface guiding users through structured data collection
 Context-aware field validation and progressive form completion
 Support for applicant, co-applicant, and hospital information capture
 Automated Document Processing with OCR
 Optical Character Recognition (OCR) for identity document extraction (Aadhar, PAN)
 Automatic population of form fields from extracted document data
 Support for multiple document formats (JPG, PNG, PDF)
 Document quality validation and legibility verification
 Intelligent Data Validation & Conflict Resolution
 PAN validation through external verification services
 Multi-source data comparison (OCR extraction vs. external API validation)
 Intelligent conflict detection and user-driven resolution workflow
 Comprehensive conflict recording for audit and compliance purposes
 Comprehensive Data Collection
 Applicant information (personal, KYC, address, insurance, banking)
 Co-applicant support with relationship tracking (optional)
 Hospital and medical treatment details
 Insurance policy validation and verification
 Employment proof and additional supporting documents
 Seamless LMS Integration
 Complete application data transmission to Loan Management System
 Conflict information and resolution history inclusion
 Full audit trail accessibility for downstream processing
 Real-time status tracking and application monitoring
 Enterprise-Grade Audit & Compliance
 Complete audit trail of all actions, modifications, and decisions
 Conflict resolution documentation with timestamps and user attribution
 Sensitive data encryption and access control
 Regulatory compliance and data retention management
System Scope
In Scope:
 Partner user authentication and authorization
 Multi-wizard application form with 4 sequential steps
 Document upload and OCR processing
 PAN validation and conflict resolution
 Data validation against external services
 Application submission workflow
 LMS integration and data transmission
 Audit trail and compliance logging
 Support for Claim Buddy and MedsCred partners
Out of Scope:
 Loan approval decision-making (handled by LMS)
 Loan disbursement processing (handled by LMS)
 Insurance policy underwriting
 Medical treatment authorization
Target Users
 Partner Loan Officers (Claim Buddy, MedsCred, and other partner organizations)
 Partner Organization Administrators
 System Administrators
 Patients/Applicants (passive participants - information collected on their behalf)
Key Business Value Propositions
 Speed: Reduce application processing time from days to hours through automation
 Accuracy: Minimize manual data entry errors through OCR and automated validation
 Compliance: Maintain complete audit trail for regulatory requirements and dispute resolution
 User Experience: Intuitive wizard-based interface reduces training requirements
 Scalability: Support multiple partners with configurable requirements
 Transparency: Real-time status tracking and conflict visibility throughout the workflow
Success Criteria
 Zero manual data re-entry for automatically extracted fields
 95%+ accuracy rate for OCR data extraction
 100% conflict detection and resolution for multi-source data
 Complete audit trail for all transactions
 Successful integration with LMS without data loss
 Support for all required partner-specific field variations
 Sub-60 minute average application completion time
Technical Considerations
 RESTful API integration with external PAN validation service
 OCR engine integration for document processing
 Encrypted storage for sensitive PAN and Aadhar data
 Database design supporting audit trail and conflict tracking
 Error handling and graceful fallback for service failures
 Performance optimization for document processing
Dependencies & Assumptions
 External PAN validation service availability and reliability
 Partner organization setup and user provisioning in LMS
